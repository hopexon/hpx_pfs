従来`padding-left`と`text-indent`を組み合わせたり`grid`や`flex`を用いて実装されていた2行目以降のインデントについて、`hanging`キーワードが全主要ブラウザで対応となった。  
`hanging`の指定により「2行目以降が指定サイズでインデント」となる。  
<br />
ただし`Chromium146(2026-03-10)`からの対応となるため、あと一年程度はプログレッシブエンハンスメント扱いとするのが穏当か。
<br />

<br />
<p className='indent-[1em_hanging]'>・indent_with_hanging<br />indent_with_hanging<br />indent_with_hanging</p>
<br />

```css
.indent__past {
  padding-left: 1em;
  text-indent: -1em;
}

.indent__past__grid {
  display: grid;
  grid-template-columns:auto 1fr;
}

// text-indent: hanging を使用したぶら下げインデント
.indent__hanging {
  text-indent: 1em hanging;
}
```
