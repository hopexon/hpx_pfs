## ◯Tools

### - Fingerboard Sheet
A training project for learning React and Next.js.  
And the best one among the Tools.  
This is the tool for helping you navigate and research the guitar,  
with functions such as

- View the note layout on the guitar fretboard
- Toggle the display of individual notes on or off
- View the interval layout around a specific fret
- Toggle the display of interval degrees on or off
- Apply various tuning and scale presets
- Use the above features after changing the root note of each string

This was inspired by ideas from guitarist friends, and is continuously being updated.  

<br />

### - dpi(s) Calculator
Three tools for checking the relationships between "dpi", "mm size" and "px".  
Mainly made for 2D designers.  
You can check the following:

- Is the dpi sufficient for the px and mm size you want to print?
- What is the maximum printable size (mm) from given px and dpi?
- How many px are needed from mm and dpi?

<br />

### - Aspect Inspector
The tool for checking the aspect ratio from images you drop.  
It's not working well (will probably be updated in the future).
